# LMLight 製品概要

**Product Overview**

---

## LMLight とは

LMLight は、**オンプレミス環境で動作するAIチャット・RAGプラットフォーム**です。

社内の機密情報を外部クラウドに送信することなく、自社環境内でAIチャット・ドキュメント検索・文字起こし・画像解析などの機能を利用できます。

---

## 解決する課題

| 課題 | LMLight の解決策 |
|------|----------------|
| クラウドAIへの機密情報の流出リスク | データが社外に出ないオンプレミス動作 |
| AIサービスの月額コスト増大 | 買い切りライセンスで継続コストを抑制 |
| 社内ナレッジが属人化 | RAGで社内文書をAIが検索・回答 |
| 複数のAIツールが乱立 | チャット・RAG・文字起こし・画像解析を一つに統合 |
| インターネットが使えない環境 | 完全オフラインで動作可能 |

---

## 主要機能

### AIチャット

- Ollama / vLLM 対応 — 多数のオープンソースモデルを利用可能
- マルチモデル切替 — 用途に応じてモデルを選択
- 会話履歴の管理

### RAG（検索拡張生成）

- 社内文書をアップロードしてBot化
- ドキュメントに基づいた正確な回答
- 対応形式: PDF, Word, Excel, PowerPoint, テキスト, Markdown

### ドキュメント生成

- AIを活用したドキュメント自動生成
- テンプレートベースの文書作成

### 文字起こし（オプション）

- 音声ファイルをテキストに変換
- Whisperモデルによる高精度認識
- 対応形式: WAV, MP3, M4A, MP4, WebM, OGG, FLAC, AAC
- 日本語・英語対応

### 物体検出（オプション）

- 画像から物体を自動検出
- YOLOv8モデルによるリアルタイム推論
- カスタム学習モデルにも対応
- 対応形式: PNG, JPG, GIF, BMP, WebP

### ユーザー管理

- 3段階のロール: ADMIN / SUPER / USER
- タグベースの共有管理
- Bot単位のアクセス制御

## システム要件

### Ollama版（macOS / Linux / Windows）

| 項目 | 最小要件 | 推奨環境 |
|------|---------|---------|
| CPU | 4コア | 8コア以上 |
| RAM | 8GB | 16GB以上 |
| ストレージ | 10GB | 50GB以上 |
| GPU | 不要（CPU動作可） | NVIDIA / Apple Silicon |

### vLLM版（Linux のみ）

| 項目 | 最小要件 | 推奨環境 |
|------|---------|---------|
| CPU | 4コア | 8コア以上 |
| RAM | 16GB | 32GB以上 |
| ストレージ | 50GB | 100GB以上 |
| GPU | NVIDIA GPU（VRAM 8GB以上） | NVIDIA GPU（VRAM 24GB以上） |
