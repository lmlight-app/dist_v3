## LMLight の強み

### 1. 完全オンプレミス・データ主権

データが社外に一切出ないため、金融・医療・製造・官公庁等の規制の厳しい業界でも安心して導入できます。

### 2. オールインワン

チャット、RAG、文字起こし、物体検出を1つのプラットフォームに統合。複数ツールの導入・管理コストを削減します。

### 3. 日本語ネイティブ

UI・ドキュメント・サポートが日本語で完結。英語が障壁になりません。

### 4. ワンコマンドインストール

```bash
# macOS
curl -fsSL https://pub-a2cab4360f1748cab5ae1c0f12cddc0a.r2.dev/scripts/install-macos.sh | bash
```

複雑なDocker構築やインフラ知識が不要。コマンド1つで環境が整います。

### 5. 柔軟なライセンス

買い切り（永続）とサブスクリプション（月額）から選択可能。クラウドAIのように「使うほど高くなる」心配がありません。

### 6. 商用サポート（SLA付き）

オープンソースのコミュニティサポートとは異なり、SLAに基づく確実なサポートを提供します。
